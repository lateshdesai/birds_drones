# Bird vs Drone Classifier

Streamlit app to classify images as **bird** or **drone** using a CNN model.

## How to deploy

1. Clone repo to GitHub
2. Make sure `app.py`, `requirements.txt`, and `runtime.txt` are present
3. Deploy on [Streamlit Cloud](https://streamlit.io/cloud)
4. The `.h5` model is automatically downloaded from Google Drive
